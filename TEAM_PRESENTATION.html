1. What We're Building

The Scenario

A rising power (Cathay) is approaching the strength of the ruling power (Columbia). History says this ends in war — the "Thucydides Trap." Participants play world leaders during the most dangerous 3-4 years in the 21st century.

The question for every participant: Can you prove history wrong? Or will you win the war it predicts?

Key Mechanics Designed

MechanicDesign Choice
Ground combatRISK dice with integer modifiers. Attacker needs ≥ defender + 1. Real dice or app dice. Moderator present.
Naval blockadeGround forces on shore (not naval superiority). Formosa requires 3-zone encirclement. 1 friendly ship deblocks to partial.
Nuclear5-tier system. 10-minute real-time authorization clock. Country-specific chains (Columbia: 3 people. Choson: leader alone).
SanctionsS-curve coalition model. West alone = modest. Add swing states = devastating. Cathay joining = game over.
IntelligencePer-individual request pools. Always returns an answer (accuracy varies 40-90%). Wrong answers look identical to right ones.
Court (democracies)AI judge. Both sides submit arguments. Common sense verdict. Moderator enforces.
CommunicationHumans talk FACE TO FACE. App only handles AI meetings and team channels.
AI assistant (Argus)Voice-first personal mentor. 7-block prompt. Remembers everything. Named for the hundred-eyed giant.

4. We Need Your Input

The following design choices have been made but we value your perspective. Please share your preference for each — either in person or via message to Marat.

Q1: Default UI Theme

The app has two themes. Which should be the DEFAULT for participants during the simulation?
Light ("Strategic Paper")
Clean, professional, newspaper feel. Better readability on phones. Current default.
Dark ("Midnight Intelligence")
Immersive, intelligence-briefing feel. Better for atmosphere. More dramatic.
User Choice
Let each participant toggle. Development cost: minimal.

Q2: Country Naming Approach

All countries have fictional names that hint at their real parallels. The names were chosen to be evocative but not offensive. Some examples: Columbia (USA), Cathay (China), Nordostan (Russia), Heartland (Ukraine), Persia (Iran).
Keep Current Names
Already developed. Participants figure out parallels naturally. Creates distance from real politics.
More Abstract
Less recognizable names. More fictional distance. Harder for participants to orient.
Real Names
USA, China, Russia. Maximum clarity. But political sensitivity risk for corporate audiences.

Q3: Role Character Names

Each role has a code name: Dealer (US President), Helmsman (Chinese Chairman), Beacon (Ukrainian President), Anvil (IRGC Commander). These are the names participants use during the SIM.
Keep Code Names Only
"I'm the Dealer." Atmospheric. Creates immersion. Current approach.
Code Name + Title
"I'm Dealer, President of Columbia." Clearer for new participants.
Code Name + Real Parallel
"I'm Dealer (think: Trump)." Maximum clarity but breaks fiction.

Q4: Map Style

The global map is a 10×20 hex grid. Currently designed as a clean, schematic map (functional, not decorative). It could be upgraded to a more atmospheric "fantasy cartography" style.
Schematic (Current)
Clean hexes, labeled, functional. Fastest to read. Lowest production cost.
Atlas/Geographic
Terrain textures, ocean shading, geographic feel. More immersive. Higher production cost.
Fantasy Cartography
Parchment, illustrated borders, compass rose. Maximum atmosphere. Highest cost.

Q5: AI Assistant Voice Character

Argus (the AI assistant) speaks to participants via voice. It should feel different from AI country participants. What personality fits best?
Sharp Colleague
Fast, direct, slightly warm. Like a smart analyst who's on your side. Current design.
Calm Authority
Measured, reassuring, slightly formal. Like a senior diplomat briefing you.
Warm Coach
Encouraging, supportive, asks questions. Like a leadership coach during a crisis exercise.

Q6: Physical Dice vs App Dice for Combat

Ground attacks and naval combat use RISK-style dice. Both players roll, moderator witnesses. Should this be physical dice in the room or digital in the app?
Physical Dice
Real dice on the table. Maximum drama. Tangible. But slower, needs moderator to record.
App Dice
Animated roll in the app. Automatic recording. Faster. But less visceral.
Both Available
Moderator chooses per situation. Physical for big moments, app for routine. Maximum flexibility.
